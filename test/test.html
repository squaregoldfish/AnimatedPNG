AnimatedPNG Example


This page shows an example of the AnimatedPNG javascript library in action. The example uses images named fish0001.png through fish0023.png. Click the various buttons below to alter the behaviour of the animation. The panel on the left will display the Javascript calls used to make the various changes.

fishAnim = new AnimatedPNG('fish', 'fish0001.png', 23, 100); fishAnim.draw(false);
